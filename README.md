# CRUD-NodeJS-Sequelize-Swagger-MySQL

Simple CRUD example with NodeJS, Sequelize, Swagger and MySQL

## Technologies:

- NodeJS Express v4.16.4
- Sequelize v4.38.0
- Swagger UI v3.20.1
- MySQL

## To run this project

- Install packages:

```batch
    npm install
```

- Run API:

```batch
    npm run dev
```

- Acess URL:

```batch
    http://localhost:3000
```
