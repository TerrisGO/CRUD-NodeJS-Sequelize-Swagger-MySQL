# CRUD-NodeJS-Sequelize-Swagger-MySQL

Simple CRUD example with NodeJS, Sequelize, Swagger and MySQL

## Technologies:

- NodeJS Express
- Sequelize
- Swagger UI
- MySQL

## To run this project

- Install packages:
  `npm install`
- Run API:
  `npm run dev`
- Acess URL:
  `http://localhost:3000`
